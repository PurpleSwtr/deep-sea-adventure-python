# Deep Sea Adventure

Консольную игра, которая представляет собой интерпретацию одноимённой настольной игры от Oink Games.
Игроки перемещается по клеточной карте, бросают кубики для определения длины хода, собирают находки разной ценности и каждый должен вернуться на подводную лодку до того, как закончится кислород.

## Установка и запуск

1. Установка:
   ```bash
   uv sync
   ```
2. Запустите игру:
   ```bash
   python src/main.py
   ```

## Структура проекта

- `src/classes/` — основные игровые объекты: дайвер, сокровище, карта, подлодка.
- `src/core/` — логика игры, конфигурация, типы данных.
- `src/presentation/` — вывод интерфейса: карта, сообщения, меню выбора (сделал чтобы не смешивать логику, и в дальнейшем можно было легко превратить проект в API где можно будет подключаться нескольким игрокам).
- `src/dependencies/` — управление внешними зависимостями.
- `src/utils/` — вспомогательные функции.

## Как играть

1. В начале игры на карту размещаются сокровища четырёх уровней ценности.
2. Игрок выбирает действие: бросить кубики для движения или развернуться.
3. Количество шагов зависит от результата броска и количества уже собранных сокровищ.
4. Каждый ход тратится кислород в зависимости от груза дайвера.
5. Найдя сокровище, можно взять его, выбросив одно из своих, или пройти мимо.
6. Цель — набрать как можно больше очков до того, как кислород опустится до нуля.

Проект находится в разработке, некоторые механики будут дорабатываться.
